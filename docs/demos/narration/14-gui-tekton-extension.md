To add a new operator surface — for example TaskRun logs or Tekton Results read-only views — start in Flask.

Add a JSON route that wraps the Kubernetes client; return stable response shapes the UI can show in tables or charts.

Add pytest with the same mock style as existing route tests.

In Vue, add or extend a Pinia store using the use API helper and team URL helper for team-scoped reads.

Add a view and router entry, then a Playwright spec so regressions catch breaks.

The extension guide in the repo lists concrete files and ideas for deeper Tekton coverage.
