# Archived shell scripts

This folder holds **retired** bash scripts that are no longer recommended for day-to-day use. They are kept for historical context or one-off migration reference.

**Do not** run these unless you know why you need them. For current entrypoints see:

- [scripts/README.md](../README.md)
- [docs/SCRIPTS.md](../../docs/SCRIPTS.md)

## Policy (same spirit as docs archive)

1. When a script is **superseded** (e.g. merged into `run-regression.sh` or replaced by a documented command), move it here instead of deleting.
2. Add a row to the table below: **script name**, **date archived**, **replacement / notes**.
3. Update any doc links that pointed at the old path.
