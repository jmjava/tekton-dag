# tekton-dag-flask

Sample Flask (pip) app for [tekton-dag](https://github.com/jmjava/tekton-dag) pipeline stacks.

- **Build tool:** pip  
- **Runtime:** flask  
- **Python:** 3.12  

Use with stack app config: `tool: pip`, `runtime: flask`, `build-command: "pip install --no-cache-dir -r requirements.txt"`.
