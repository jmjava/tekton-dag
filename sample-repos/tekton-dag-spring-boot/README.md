# tekton-dag-spring-boot

Sample Maven Spring Boot app for [tekton-dag](https://github.com/jmjava/tekton-dag) pipeline stacks.

- **Build tool:** maven  
- **Runtime:** spring-boot  
- **Java:** 21  

Use with stack app config: `tool: maven`, `runtime: spring-boot`, `build-command: "mvn -B clean package -DskipTests"`.
